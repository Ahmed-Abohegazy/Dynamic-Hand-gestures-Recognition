IMPORTANT NOTES 
### The technologies used:
Python 3.7  
TensorFlow GPU V2
KERAS 
OpenCV

### Description to read before you begin:
-training-sample.PY: due to  the limitation of the processing power this script is used to extract wanted gestures and the num of samples per gesture 
-real_time.py: is used to start the cam and predict gesture 
-controll_commands.py: the script is used to link between the gesture predicted and the command to get applied 

###how to run the code to control spotify :
-run real_time.py
-open spotify (note make sure spotify is selected and in full screen mode)
